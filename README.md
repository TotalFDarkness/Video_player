# Видеоплеер
Проект представляет собой минималистичный видеоплеер с активным функционалом.

Ознакомиться с плеером, можно по этой [ссылке](https://totalfdarkness.github.io)

## Функционал
- Контроль воспроизведения (play/pause).

- Управление громкостью (mute/unmute)

- Включение полноэкранного режима.

Чтобы выйти из этого режима, прожмите `esc` на клавиатуре.
